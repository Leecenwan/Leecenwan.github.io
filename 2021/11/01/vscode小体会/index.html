前言

听江湖上说vscode有“一器破万法”的美称，上课学习上班摸鱼干啥都行的神器。是因为它强大的插件扩展，最重要它还是开源，免费。
恰巧最近写博客我也下了这个编译器，感受一下它的快乐~

  1. 下载
    首先vscoede的下载和安装还是比较简单的。大家去Download然后不同的系统大家自行选择就行了。
  2. 插件
    大家安装完打开以后的页面都是英文的，对于我来说用起来比较困难。首先我们要去左边的扩展功能(Ctrl+Shift+x)里面搜索chinese→找到中文简体→Install，下载在重启vscode之后就是中文页面了。
  • 主题更改

在文件→自选项→颜色主题。我还是比较喜欢快捷键Ctrl+K+t就会出来了

大家可以一个个试，如果还没有适合你的，可以下拉到最后有个安装其他颜色主题，相信总会有适合你的。

  • Beauty
    这个是我最近常用的插件，超级好用。因为博客里面很多的样式代码都压缩了，打开简直密密麻麻看的你头皮发麻，这个插件就能很好的让代码还原。使用也超级方便，全选代码然后Ctrl+Shift+b就会还原了
  • markdown侧边预览
    很多人写文章可能都在用typora,我比较懒，写代码和文章都用vscode了。这里有个侧边实时预览功能我好喜欢。按F1就会弹出来

    选第一个，就会打开了。
  • guides

    显示代码对齐的辅助线，有时候我写div的时候也会让它们在同一条线上，不然div嵌套多了不好找…这个其实对于强迫症的同学还是蛮有意思的~
  • HTML Snippets
    HTML代码提示
  • HTML CSS Support
    智能提示CSS样式
  • CSS Peek
    可以在HTML中通过CSS id或则class来定位到其定义。
  1. 摸鱼插件
  • 小霸王

    哈！每天上班小霸王,下班冲饭堂~
  • cloudmusic
    网易云有的感觉它都有
  • NBA
    球迷必备，看球嘛顺便上个班挺正常的=-

END
